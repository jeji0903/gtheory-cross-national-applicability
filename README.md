# G-Theory Cross-National Applicability Study

This repository contains R code for the simulation and empirical analyses presented in the manuscript.

## Contents

- `simulation.R`: Generates simulated datasets under different variance conditions, computes G-theory variance components and D-study reliabilities, and produces publication figures (Figures 1–5).
- `analysis.R`: Performs G-theory D-study and CFA measurement invariance analyses on the PISA dataset.

## Requirements

- R (>= 4.0)
- Packages: `lme4`, `dplyr`, `tidyr`, `purrr`, `ggplot2`, `lavaan`, `semTools`, `readr`

## Notes

- The PISA dataset is not included due to data restrictions.
- Code is fully reproducible given access to the dataset.
